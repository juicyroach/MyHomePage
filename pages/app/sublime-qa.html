Sublime Q&A by Juicyroach

Posted on May 19, 2015 at 11:00 PM

這篇記錄了一些我或我身邊的人遇到關於Sublime的問題與解決方式。偷懶把一些常用的外掛也放在這裡好了...


如何安裝外掛!?

安裝 Package Control (總管理所有外掛的外掛)

  • 到 https://sublime.wbond.net/Package%20Control.sublime-package 下載檔案
  • 開啟 sublime 上方 Preferences => Browse packages 會跳出檔案總管
  • 在跳出的檔案總管位置 選擇到 上一層 資料夾位置 ( *\AppData\Roaming\Sublime Text 2)
  • 進入Installed Packages資料夾內，將下載下來的檔案放入
  • 重新啟動Sublime，就安裝成功! (可在工具列的Preferences選單中找到Package Control 選項)
  • [有了Package Control即可以快速安裝任何插件]

中文會亂碼耶!?

安裝ConvertToUTF8外掛!

  • 在Sublime頁面案下ctrl+shift+p
  • 輸入package control: install package，等待數秒 [開啟package control外掛]
  • 在出現的框框打上convertToUTF8，按下enter!
  • 重新啟動Sublime，即可完成!

如何用滑鼠區域性的選取想要的資料!?

請使用shift+滑鼠右鍵，框選想要的部分!

  • 在Sublime頁面案下ctrl+shift+p
  • 輸入package control: install package，等待數秒 [開啟package control外掛]
  • 在出現的框框打上convertToUTF8，按下enter!
  • 重新啟動Sublime，即可完成!

如何顯示Space空格數與Tab數量!?

  • 上方工具列 → Preference → Settings-User 點選會跳出一個使用JSON寫成的設定檔頁面
  • 大括弧內加入 "draw_white_space": "all" 即可完成!!
  • [Tab會使用-表示,Space會使用.表示]

如何快速將多行合併成一行!?

  • 反白多行，或用ctrl+左鍵點選各行某處
  • ctrl+j

如何快速刪除空白行!?

運用正規表示式搜尋

  • Crtl+H (也可以點選工具列Find → replace)
  • alt+r (或是點選最左邊的按鈕) 開啟正規表示式搜尋
  • find欄位填入^\n replace欄位空白不要填 選擇replace all即可完成!

有沒有辦法快速改html的標籤大小寫呢?!

運用正規表示式搜尋

  • Ctrl+f
  • alt+r (或是點選最左邊的按鈕) 開啟正規表示式搜尋
  • 打入 <(/?\w+) 找出全部的標籤並有框框標註
  • alt+enter 框起全部找到的結果
  • ctrl+k+u 全部改大寫 (ctrl+k+L 全部改小寫)

如何選擇顯示模組?

Sublime有許多模組，開啟不同類型的時候會自動判斷副檔名，在新建檔案或有某時想更換顯示類型的話，可以點選右下角更換檔案型別，Sublime會自動的幫你上好顏色唷!

我不喜歡黑色，可以換顯示的主題嗎?

請至上方工具列 Preferences → Color Scheme內預覽更換!

我想要format JavaScript檔案!

請安裝jsFromat

  • ctrl+shift+p
  • 輸入package control: install package
  • 在出現的框框打上jsFromat，按下enter! 等待左下角顯示安裝完成後重開Sublime即可!
  • 預設format鍵為ctrl+alt+f，可在Preferences → package settings → jsFormat → key blindings-Default裡面修改!

Syntax模組有好多用不到，我想移除讓我比較快能選到我要的!

有兩種方法可以達到

◎ 眼不見為淨大法

  • 工具列Preferences → Settings-User
  • 在JSON檔案內加入 "ignored_packages":[" Python",“想要看不到的Syntax名稱”]
  • 儲存後可以完成

◎ 斬草除根大法

  • 工具列Preferences → Browse Packages
  • 直接刪除不想要用的Syntax !
  • 重開Sublime後即可完成

怎麼打開旁邊的side bar讓我能快速找到檔案呢?

  • 工具列view → side bar → show side bar (hide side bar)
  • 或著覺得看旁邊很麻煩，也可以用ctrl+p用打部分關鍵字快速跳換!
